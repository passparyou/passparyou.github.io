Blockchain

A block chain is a transaction database shared by all nodes participating in a system based on the Bitcoin protocol. This is the main innovation introduced in the Bitcoin protocol. It guarantees that a single data added in this transaction database has to be confirmed by distributed node. Once confirmed, that data can't be modified, because all data subsequently added should be also modified in a consistent manner. That's something impossible.
